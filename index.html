Open source secret scanner for AI coding tools

Your AI assistant shouldn't know your secrets.

01 — The problem

The security gap nobody talks about

Paste and pray

Developers paste .env files, connection strings, and API keys into AI tools dozens of times a day. No inspection layer exists between your clipboard and a third-party API.

DLP can't read prompts

Network-level DLP can block api.anthropic.com entirely, but it can't parse a JSON request body to find the AWS key buried in message 47 of a conversation.

Git scanning is too late

GitGuardian catches secrets after commit. But when you paste a key into Claude, it's already on Anthropic's servers. There's no git push to intercept.

02 — Setup

Two lines. That's it.

Prompts with secrets are blocked. Tool inputs (Bash, Write, Edit) are redacted automatically — real credentials saved to .env, placeholders sent to Claude. No proxy, no certificates.

Works with any AI tool — Cursor, Copilot, direct API calls. Transparent interception via mitmproxy.

04 — Enforcement

Four modes. Your call.

Redact

Replace & rewrite

Replaces secrets with env var placeholders, saves real credentials to .env, and injects a system instruction. The AI writes secure code using environment variables automatically.

Block

Reject & explain

Returns a clear error. The developer removes the secret and retries. Ten seconds of friction that prevents a breach.

Warn

Forward & log

Sends the request but logs a warning. Use this when rolling out aigate to understand exposure before enforcing.

Audit

Silent observe

No blocking, no warnings. Writes every detection to a JSON audit log. Perfect for proving compliance to auditors.

05 — Redact mode

Secrets in, env vars out.

Redact mode rewrites requests before they reach the AI. Secrets are replaced with named placeholders. Real credentials are saved to a local .env file. A system instruction tells the AI to use environment variables. The AI never sees the real values.

06 — Technical details

Under the hood

Codebase
~900 lines of Python. Read the whole thing in 20 minutes.
Performance
Typical prompts scan in 0.005ms. Even 1MB of text scans in 67ms.
Detection
Regex + Shannon entropy. Deterministic, auditable, no model downloads.
Allowlist
Suppress known false positives with fnmatch wildcard patterns.
Audit log
JSON. ISO 8601 timestamp, provider, rule, redacted match, action.
Telemetry
None. Nothing leaves your machine. No accounts, no analytics.
